# up

Simplified deployments.

## Setup

1.  Install up: `npm i -g @karimsa/up`
2.  Log into your cloud provider: `up login -p [provider]` (or just `up login` for DigitalOcean)

## Usage

**Deploying application:** `up`
**Look at logs:** `up logs`
